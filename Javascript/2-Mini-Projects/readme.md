# JavaScript Mini Projects

A collection of mini projects demonstrating core JavaScript concepts through practical examples.

## Overview

This repository contains small, focused projects that illustrate fundamental JavaScript programming concepts. Each project is designed to reinforce understanding through hands-on practice. There is a variety of topics covered, from basic syntax to more advanced features, with examples and with a challenge for each topic.

## Projects

### 1. DOM Manipulation

- Dynamic element creation and modification
- Event handling
- CSS manipulation via JavaScript

### 2. Asynchronous JavaScript

- Promises and async/await
- Fetch API
- Error handling

### 3. ES6+ Features

- Arrow functions
- Destructuring
- Spread/Rest operators
- Template literals

### 4. Array Methods

- map, filter, reduce
- forEach, find, some, every
- Practical data transformation

### 5. Object-Oriented JavaScript

- Constructor functions
- Classes and inheritance
- Prototypes

## Getting Started

1. Clone the repository
2. Navigate to any project folder
3. Open `index.html` in your browser
4. Check the console for outputs

## Purpose

These projects serve as:

- Learning exercises for JavaScript fundamentals
- Reference implementations
- Building blocks for larger applications
